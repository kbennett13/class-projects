import_project.html, import_project.php - Allows the user to access an existing project.

import_music.php, import_music2.php - Allows the user to add a music file.

counts.php - The user interface.

figure.js - Draws the figure on the canvas. Contains event handlers. Figure code adapted from Angel and Shreiner’s Interactive Computer Graphics: A Top-Down Approach with WebGL (7th edition).

xml_handlers.js - Parses and writes XML to and from the user’s project for saving and animation.
